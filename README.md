# Lab 5: JavaScript Basics – Conditions, Loops, Functions, Objects, and Arrays

**Objective:**  
To understand JavaScript syntax, conditional statements, loops, functions, and arrays using HTML and `<script>`.

---

## Activities

### **Activity 1: Add Two Numbers**
**Purpose:** Perform arithmetic operations using JavaScript and display the result on a webpage.  

**Concepts Used:**  
- Variables (`let`)  
- Type conversion using `Number()`  
- DOM manipulation  
- Arithmetic operators  

**Description:**  
The program takes two numbers from the user and adds them when the button is clicked. The result is displayed dynamically on the webpage.

---

### **Activity 2: Login System**
**Purpose:** Simulate a simple login validation.  

**Concepts Used:**  
- Logical AND (`&&`)  
- String comparison  
- Conditional statements  

**Description:**  
The system validates predefined username and password and displays either a success or failure message.

---

### **Activity 3: Array**
**Purpose:** Learn how to create and manipulate arrays in JavaScript.  

**Problem:**  
Create an array of 5 student names. Perform the following:  
- Display all names  
- Add one new student  
- Remove the last student  
- Display the updated array  

**Description:**  
Students are stored in an array, and operations like adding and removing elements are demonstrated with dynamic display on the webpage.

---

### **Activity 4: Object**
**Purpose:** Understand how to create and access objects.  

**Problem:**  
Create a student object with:  
- Name  
- Age  
- Department  

Display the student information on the webpage.  

**Description:**  
The object’s properties are accessed and displayed using DOM manipulation.

---

## Tasks

### **Task 1: Basic Calculator Enhancement**
**Requirements:**  
- Add buttons for subtraction, multiplication, and division.  
- Display an error message if inputs are empty.  
- Change the result color dynamically (green for valid result, red for error).  

---

### **Task 2: Number Checker**
**Requirements:**  
- Check whether a number is even or odd.  
- Display messages in green for positive numbers and red for negative numbers.  
- Show an alert if input is empty.  

---

### **Task 3: Array-Based Web Page**
**Requirements:**  
- Create an array of 6 numbers.  
- Calculate and display the sum of all numbers.  
- Find and display the largest number.  
- Display the array in reverse order.  
- Show all results on the webpage (not console).  

---

### **Task 4: Array of Objects**
**Requirements:**  
- Create an array of 3 student objects (name and marks).  
- Display all students on the webpage.  
- Calculate and display the average marks.  
- Find and display the topper student.  
- Use loops to iterate over the array of objects and apply CSS styling for proper display.

---

## **Summary**
This lab introduces fundamental JavaScript concepts, including variables, arithmetic operations, conditional statements, loops, arrays, and objects. Through hands-on activities and tasks, students learn to dynamically manipulate and display data on web pages using HTML and `<script>`.
